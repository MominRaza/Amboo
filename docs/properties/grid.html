Grid

Grid can have 8 columns without specifing device size. But with device size specification there are some downgrandes and upgrades you can make.

By default you can add grid-2, grid-3 ... grid-8.

With sm-mobile you can only add grid-sm-mobile-2, grid-sm-mobile-3, grid-sm-mobile-4

With mobile you can only add grid-mobile-2, grid-mobile-3 ... grid-mobile-6

With tablet you can only add grid-tablet-2, grid-tablet-3 ... grid-tablet-8

With sm-laptop you can only add grid-sm-laptop-2, grid-sm-laptop-3 ... grid-sm-laptop-10

With laptop you can only add grid-laptop-2, grid-laptop-3 ... grid-laptop-12

If you adding a big device classes than you must be need to add small classes otherwise in small screensize grid will be automatically 1.

Example: first add big screen size classes like grid-laptop-8, now add small screen size classes like grid-tablet-6 now add more small screen class like grid-mobile-3.

Note: grid-sm-mobile works diffrently
